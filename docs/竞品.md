# Feed The Pit 竞品与来源台账

检查日期：2026-07-02

## 结论

Feed The Pit 目前没有成熟英文 wiki 竞品。首发机会不在“复制大型数据库”，而在抢占高意图基础页：wiki、guide、play online、Steam、download、cards、monsters、walkthrough、Act 1、ending、achievements、rendering fixes、Discord、mobile safety。

需求中同时出现 Steam 与 itch.io。调研后必须分开处理：

- Steam：Curious Fox Sox 的 story-driven investigative horror，是本站攻略主对象。
- itch.io：mooncrustgames 的 HTML5 dark incremental，是本站 `/play-online` 自托管对象。
